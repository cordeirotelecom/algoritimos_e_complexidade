📚 Sobre o Algoritmo Atual

Bubble Sort: Algoritmo simples que compara elementos adjacentes repetidamente, fazendo os maiores "borbulharem" para o final do array.

Melhor caso: O(n) - array já ordenado
Caso médio: O(n²) - elementos aleatórios
Pior caso: O(n²) - array inversamente ordenado
